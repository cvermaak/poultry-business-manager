# AFGRO Poultry Manager - Design Notes

## Reference: Agrarium Farming Theme Analysis

### Color Palette Extracted:
- **Primary Green**: #7CB342 (lime/yellow-green - used in logo, buttons, accents)
- **Dark Background**: #1A1A1A / #2D2D2D (dark sections)
- **Accent Yellow/Gold**: #D4A84B (wheat/harvest gold - decorative text)
- **Red Accent**: #E53935 (used sparingly for emphasis)
- **Cream/Off-white**: #F5F5DC / #FAFAF5 (light backgrounds)
- **Text Dark**: #333333 (body text on light backgrounds)
- **Text Light**: #FFFFFF (text on dark backgrounds)

### Typography:
- **Headings**: Script/handwritten style font (decorative, organic feel)
- **Subheadings**: Serif or elegant sans-serif
- **Body Text**: Clean sans-serif for readability
- **Accent Text**: Cursive/script for labels like "About Farming", "Experience"

### Design Elements:
- Leaf/plant icons as brand marks
- Full-width hero images with overlay text
- Split layouts (image + content side by side)
- Dark sections with light text for contrast
- Outlined buttons with hover fills
- Decorative wheat/plant graphics
- Timeline/history sections
- Gallery grids
- Team member cards
- Testimonial carousels

### Key Characteristics:
1. **Organic/Natural Feel**: Earthy colors, plant imagery
2. **High Contrast**: Dark sections alternating with light
3. **Script Typography**: Adds warmth and personality
4. **Full-bleed Images**: Immersive photography
5. **Yellow-Green Primary**: Fresh, agricultural, growth-oriented
6. **Gold Accents**: Premium, harvest, success

## AFGRO Poultry Manager Theme Plan

### Adapted Color Palette:
- **Primary**: #7CB342 (Agrarium green) - buttons, links, active states
- **Primary Dark**: #558B2F (darker green for hover states)
- **Secondary**: #D4A84B (harvest gold) - accents, highlights
- **Background Light**: #FAFAF8 (warm off-white)
- **Background Dark**: #1E2A1E (dark green-tinted black for contrast sections)
- **Surface/Cards**: #FFFFFF (clean white)
- **Text Primary**: #2D3A2D (dark green-tinted black)
- **Text Secondary**: #5D6D5D (muted green-gray)
- **Success**: #4CAF50 (green)
- **Warning**: #FF9800 (amber/orange)
- **Error**: #E53935 (red)
- **Border**: #E0E4E0 (light gray-green)

### Typography Plan:
- **Display/Logo**: Playfair Display or Cormorant (elegant serif)
- **Headings**: Poppins or Montserrat (modern, clean)
- **Body**: Inter or Open Sans (highly readable)
- **Accent Labels**: Caveat or Dancing Script (handwritten feel for section labels)

### Logo Concept for AFGRO:
- Incorporate chicken/poultry silhouette
- Leaf or growth element
- "AFGRO" in bold, modern typography
- Tagline: "Poultry Manager" in lighter weight
- Color: Primary green with gold accent

### Dashboard Adaptations:
- Sidebar: Dark green (#1E2A1E) with white text
- Cards: White with subtle shadow
- Stats: Green icons with gold accent numbers
- Tables: Clean with green header row
- Buttons: Filled green primary, outlined secondary
